## 💼 Featured Projects

<div align="center">

### 🏭 Bharat Modules — B2B Procurement Platform
<sub>`Production` · `Live Traffic`</sub>

</div>

> **Problem:** Indian manufacturers and enterprise buyers lacked a trusted, structured channel for quotations and order management.
> **Solution:** A production REST backend powering the procurement lifecycle end-to-end.

| Aspect | Details |
|--------|---------|
| **🏗️ Architecture** | Node.js + Express REST API, PostgreSQL (Supabase) with Row-Level Security, deployed on Railway |
| **🔐 Security** | JWT auth + **role-based access control** (buyer / manufacturer / admin), OTP email verification, password reset via Resend, rate limiting, CORS, Helmet |
| **⚙️ Key Features** | RFQ (Request for Quotation) workflows, order management with end-to-end status tracking, Cloudinary file attachments |
| **📈 Production Readiness** | Hardened middleware stack, RLS-enforced data isolation, live production traffic |
| **🧰 Stack** | `Node.js` `Express` `PostgreSQL` `Supabase` `JWT` `Cloudinary` `Firebase` `Railway` |

### 🚨 ResoNet — AI Disaster Resource Allocation System
<sub>`🥇 Hackathon Winner` · `Multi-Agent` · `Real-Time`</sub>

</div>

> **Problem:** Emergency services need fast, explainable resource allocation during disasters.
> **Solution:** A multi-agent system that simulates disaster scenarios and allocates resources in real time — with AI-generated explanations for every decision.

| Aspect | Details |
|--------|---------|
| **🏗️ Architecture** | FastAPI backend + **WebSocket server** for real-time coordination, multi-agent decision engine, SQLite persistence |
| **🤖 AI Layer** | **Groq API** generates plain-language rationale per allocation; **Random Forest** trained on **260K+ earthquake records** for damage-zone classification |
| **🗺️ Frontend** | React 19 + Leaflet live map dashboard visualizing allocations as they happen |
| **⚡ Engineering Challenge** | Keeping agent state consistent and explanations synchronized under real-time WebSocket load |
| **🏆 Outcome** | **1st place** — Adhigamya Udbhav Hackathon 2026, built in 20 hours |
| **🧰 Stack** | `Python` `FastAPI` `WebSockets` `React 19` `Leaflet` `Scikit-learn` `Groq API` `SQLite` |
